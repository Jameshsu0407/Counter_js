# 使用說明
jQuery CDN

    <script src="https://ajax.googleapis.com/ajax/libs/jquery/3.4.1/jquery.min.js"></script>

Counter CDN

    <script src="https://jameshsu0407.github.io/Counter_js/counter.js"></script>

添加屬性

    data-ct-number="數字"
    data-ct-duration="時間"
